# VtM5e_Dice-Statistics
[R] [VtM5e] A small collection of functions for dice statistics in Vampire The Masquerade 5. Edition.

V5stats is a Monte Carlo Simulation for calculating the likelihood of certain dice rolls. 
It includes the chance of messy criticals and bestial failure through Hunger Dice as well as the correct number of successes by rolling one or more 10s.
